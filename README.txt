This plugin integrates a 5Stelle Booking reservation form.

== Description ==

Integrate 5Stelle Booking reservation painlessly in your website.

The plugin is developed using AngularJs, Moment.js and Lodash.
JavaScript must be activated for the plugin to work.

== Installation ==

As simple as it gets:

1. Upload the plugin
2. Activate it
3. Configure it by setting the option fields under Settings -> Visa 5Stelle Booking
4. Place [v5b_display_form] shortcode wherever you want
